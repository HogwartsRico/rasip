# rasip
Speak ip address  on Raspberry PI 


# Requirements
`awk,sed` extra utils

`mplayer` sound palyer

# Install
`curl -L https://raw.githubusercontent.com/rockagen/rasip/master/rasip.sh -o rasip.sh`

`chmod u+x raspi.sh`

# Run
`./raspi.sh`

or enable auto after system boot

`mv raspi.sh /etc/init.d`

happy enjoy!
